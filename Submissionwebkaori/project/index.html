Penjelasan Arduino Uno

Arduino Uno adalah papan mikrokontroler berbasis ATmega328P, dirancang untuk mempermudah pengembangan perangkat keras dan perangkat lunak dalam bidang elektronika. Papan ini menjadi pilihan populer untuk pemula maupun profesional karena kemudahannya dalam digunakan.

Arduino Uno memiliki 14 pin input/output digital (di mana 6 di antaranya dapat digunakan sebagai output PWM), 6 pin input analog, osilator kristal 16 MHz, koneksi USB, jack daya, header ICSP, dan tombol reset.

Fungsi Utama Arduino Uno

  • Kendali Perangkat Elektronik: Mengontrol perangkat seperti LED, motor, atau relay dengan mudah.
  • Pengumpulan Data Sensor: Menghubungkan sensor suhu, kelembapan, cahaya, dan lainnya untuk mengumpulkan data lingkungan.
  • Prototipe Cepat: Memungkinkan pengembangan prototipe sistem otomatisasi dengan cepat.

Spesifikasi Arduino Uno

Parameter Detail
Mikrokontroler ATmega328P
Input Voltage 7-12V
Digital I/O Pins 14 (6 sebagai PWM)
Analog Input Pins 6
Flash Memory 32 KB (0.5 KB digunakan oleh bootloader)
Clock Speed 16 MHz

Komponen Pendukung

Beberapa komponen yang sering digunakan bersama Arduino Uno:

  • Sensor DHT11 (suhu dan kelembapan)
  • Motor Servo
  • Modul RFID
  • Shield Ethernet
